Berikut adalah daftar perangkat standar yang dibutuhkan untuk memulai layanan.

### Perangkat Utama WiFi
| Perangkat | Fungsi | Contoh Merk |
| :--- | :--- | :--- |
| **Router Board** | Mengatur bandwidth & manajemen user. | MikroTik |
| **Access Point** | Menyebarkan sinyal WiFi ke client. | TP-Link, Ubiquiti |
| **Kabel LAN** | Transmisi data antar perangkat. | Belden Cat5e/6 |

### Perangkat Utama CCTV
1. **Kamera (IP Cam / Analog):** Sesuaikan dengan kebutuhan (Indoor/Outdoor).
2. **DVR/NVR:** Alat perekam video dari kamera.
3. **Harddisk:** Khusus untuk CCTV (Skyhawk/Purple) agar tahan menyala 24 jam.
4. **Power Supply Center:** Untuk menyuplai listrik ke banyak kamera sekaligus.

### Peralatan Teknisi
* Crimping Tool
* LAN Tester
* Tangga Teleskopik
* Bor Listrik
